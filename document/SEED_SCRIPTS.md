# SEED_SCRIPTS

샘플 데이터/템플릿 생성 스크립트 목록입니다.

## 주요 스크립트

- `scripts/seed-templates.ts` — 기본 템플릿 시드
- `scripts/seed-sample-course.ts` — 샘플 코스/강의/일정 시드
- `scripts/create-sample-course.ts` — 단일 샘플 코스 생성
- `scripts/create-sample-courses.ts` — 다수 샘플 코스 생성
- `scripts/create-instructors.ts` — 강사 샘플 생성
- `scripts/create-users-by-role.ts` — 역할별 사용자 생성
- `scripts/update-instructor-profile.ts` — 강사 프로필 업데이트
- `scripts/update-password.ts` — 비밀번호 업데이트
- `scripts/backfill-instructor-users.ts` — 강사 사용자 매핑 보정
- `scripts/check-sample.ts` — 샘플 데이터 점검

> 실행 방식은 `package.json` 또는 스크립트 내부 주석을 참고하세요.
